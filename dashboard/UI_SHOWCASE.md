## 🎬 使用演示

### 场景 1：简单查询（快速成功）

**输入**：`SQL注入 payload`

**流程**：
```
第1轮 → Search → Detect ✅ 成功 → 直接返回结果
```

**耗时**：~150ms

---

### 场景 2：CVE 编号（需要重写）

**输入**：`CVE-2017-10271`

**流程**：
```
第1轮 → Search → Detect ❌ 失败
       → Diagnose: "Query Drift - CVE编号不够语义化"
       → Rewrite: "CVE-2017-10271 WebLogic RCE"

第2轮 → Search → Detect ✅ 成功 → 返回高质量结果
```

**耗时**：~300ms

---

### 场景 3：过于宽泛（多轮迭代）

**输入**：`Windows漏洞`

**流程**：
```
第1轮 → Detect ❌ → Diagnose: "Too Broad"
       → Rewrite: "Windows提权漏洞"

第2轮 → Detect ❌ → Diagnose: "Still Broad"
       → Rewrite: "Windows UAC绕过技术"

第3轮 → Detect ✅ → 返回精准结果
```

**耗时**：~450ms

## 🎨 视觉特效一览

### 颜色方案
- **主色调**：`#667eea` - `#764ba2`（蓝紫渐变）
- **强调色**：`#00ffff`（霓虹青色）
- **成功色**：`#11998e` - `#38ef7d`（绿色渐变）
- **失败色**：`#eb3349` - `#f45c43`（红色渐变）

### 动画效果
- **霓虹脉冲**：`neon-pulse` - 输入框、激活步骤
- **渐变流动**：`gradient-shift` - 按钮背景
- **滑入动效**：`slide-in` - 卡片出现
- **发光呼吸**：`pulse-glow` - 图标闪烁

### 交互反馈
- **悬停放大**：卡片 `scale(1.05)`
- **点击缩小**：按钮 `scale(0.98)`
- **阴影扩散**：悬停时 `box-shadow` 增强
