# Evo-PentestRAG - 项目概述

## 🎯 项目简介

**Evo-PentestRAG** (Evolutionary Penetration Testing with Retrieval-Augmented Generation) 是一个创新的渗透测试知识库系统，结合了多源数据爬取、RAG（检索增强生成）和自适应学习机制，专为智能化渗透测试场景设计。

### 核心理念

不同于传统的知识检索系统，Evo-PentestRAG 引入了 **"功能对齐"（Functional Alignment）** 的概念——评估知识的价值不是看语义相关性，而是看它是否能帮助 Agent **真正拿到 Shell**。

---

## 🌟 核心特性

### 1. 多源知识爬取
- 支持 **10+ 公开安全数据源**（CSDN、GitHub、MITRE ATT&CK、CVE、NVD 等）
- 动态爬虫 + 静态数据库同步双轨并行
- 可扩展的爬虫框架，易于添加新数据源

### 2. 智能RAG系统
- **自适应检索**：基于攻击上下文动态调整检索策略
- **价值重排序**：使用 Cross-Encoder 模型对知识进行"实战价值"排序
- **失败反思**：System 2 思考机制，从失败中学习并自动改写查询

### 3. 知识自进化
- **HER (Hindsight Experience Replay)**：从攻击日志中自动提取训练样本
- **后果驱动标注**：基于攻击成功与否自动打标签
- **经验积累**：成功的攻击经验自动沉淀到知识库

### 4. 可视化管理
- Web Dashboard 实时监控爬取进度
- 攻击轨迹可视化展示
- 模型训练实时反馈

## 🔬 创新点

### 1. 全球首个"后果驱动"的渗透知识数据集
传统 RAG 评估看"词汇匹配"，我们看"攻击成功率"。

### 2. System 2 反思机制
让 AI 能够发现"为什么这次搜索失败了"，并自动调整策略。

### 3. Hard Negative Mining
专门训练模型识别"看似有用，实则无用"的过时 Exploit。

### 4. 多模态知识对齐
通过 VLM 将文章中的截图转为文字，消除信息盲区。

## 📊 技术栈

### 数据层
- **爬虫框架**: Requests + BeautifulSoup + DrissionPage
- **数据存储**: JSON + RAGFlow 向量数据库
- **数据格式**: STIX 2.1, JSON, XML, CSV

### RAG层
- **向量检索**: RAGFlow API
- **重排序模型**: Cross-Encoder (ms-marco-MiniLM-L-6-v2)
- **LLM集成**: DeepSeek / OpenAI API

### 学习层
- **日志分析**: Python + JSONL Parser
- **模型训练**: PyTorch / Sentence-Transformers
- **评估框架**: 自研 Evaluator

### 可视化层
- **后端**: Flask
- **前端**: Bootstrap + Chart.js + jQuery
- **实时通信**: Server-Sent Events (SSE)
